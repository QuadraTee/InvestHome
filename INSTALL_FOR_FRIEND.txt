INVESTHOME FINANCE TRACKER v2.6.0 - EASY INSTALL
=================================================

1) Copy the zip to your Linux server/LXC.

2) Unzip it:

   unzip finance_tracker_v2.6.0_trading212_modern_ui.zip
   cd finance_tracker_v2.6.0_trading212_modern_ui

3) Create a Python virtual environment:

   python3 -m venv venv
   source venv/bin/activate

4) Install requirements:

   pip install -r requirements.txt

5) Create the environment file:

   cp .env.example .env

6) Start the app:

   python3 app.py

7) Open it in a browser:

   http://SERVER-IP:5000

TRADING 212 OPTIONAL SETUP
==========================

Edit .env and add your Trading 212 credentials:

   TRADING212_API_KEY=your_api_key_here
   TRADING212_API_SECRET=your_api_secret_here

Restart the app after changing .env.

UPDATING FROM AN OLDER VERSION
==============================

Keep these files:

   finance.db
   .env

Overwrite everything else with the new version, then restart:

   pkill -f "python3 app.py"
   source venv/bin/activate
   python3 app.py

RESETTING THE APP
=================

To start from fresh data:

   rm finance.db
   python3 app.py

BACKUP
======

Before updating:

   cp finance.db finance_backup_$(date +%F).db
